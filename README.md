# Learning Blog

A personal learning blog and notes website built with Next.js, TypeScript, Tailwind CSS, and MDX.

## Features

- Home, About, blog list, and blog detail pages
- MDX-powered notes in `content/posts`
- Tags and categories
- Client-side search
- Responsive documentation-style layout

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
